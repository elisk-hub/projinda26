# Smart To-Do App

## Projektbeskrivning
Detta projekt syfte är att utveckla en webbapplikation som hjälper användare att organisera och planera sina uppgifter på ett smart sätt. 

Till skillnad från en vanlig to-do-lista fokuserar denna applikation på att:
- prioritera uppgifter automatiskt
- föreslå vad användaren bör göra idag
- ge en enkel översikt över veckan

Projektet kombinerar regelbaserad logik med AI-stöd för att skapa ett enklare rekommendationssystem.

---

## Funktionalitet

### Grundfunktioner
Applikationen ska innehålla följande grundläggande funktioner:

- Skapa uppgifter  
- Ange titel och beskrivning  
- Sätta deadline  
- Välja prioritet (låg, medel, hög)  
- Markera uppgifter som klara  
- Visa alla uppgifter i en lista  

---

### Smarta funktioner
Applikationen kommer även att innehålla funktioner för att hjälpa användaren att planera:

- Automatisk sortering av uppgifter
- Föreslå ordning mellan uppgifter  
- Visa vilken uppgift som bör göras först  
- Visa en sektion: **“Detta borde du göra idag”**  
- Generera en enkel veckoplan  

Dessa funktioner baseras på:
- Regelbaserad logik (deadline + prioritet)
- AI-stöd via ett externt API

---

### Visualisering
För att göra planeringen tydlig för användaren:

- Visa dagens uppgifter  
- Visa veckans uppgifter  

---

## Teknisk lösning

Projektet implementeras som en webbapplikation:

- Frontend:React + TypeScript  
- Backend:Python 
- API:LLM API (för AI-funktioner)  
- Databas:SQLite
